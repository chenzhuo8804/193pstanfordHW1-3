This is the Assignment3 for iPad and iPhone development by Stanford University (CSE 193p http://www.stanford.edu/class/cs193p/cgi-bin/drupal/) . The product of this assignment is a graphing calculator. Detailed information can be found in the pdf file in root directory

Asst3 implemented a graphing calculator, which can draw graph of the program in the stack.The calculator uses a stack to manage it operands, so if you want to calculator 3+5 , you will first press 3,press enter, then press 5, then press +. The Enter button would press 3 into the stack and the + would pop the operands off the stack.

Demo, press a , press sin then press Graph would generate the graph of sin(a)
